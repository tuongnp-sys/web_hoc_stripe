# Audio assets

Place MP3 files here. The game runs silently if files are missing.

## Background music (per layer)

- `bg-layer-1.mp3` … `bg-layer-7.mp3` — one track per layer
- `bg-music.mp3` — fallback when a layer track is missing

## Sound effects

- `sfx-collect.mp3`, `sfx-hit.mp3`, `sfx-shockwave.mp3`
- `sfx-ten.mp3` — bright pickup chime when collecting mindfulness (Scripture)
- `sfx-duc.mp3` — dull sad tone when hitting negative thoughts
- `sfx-layer-up.mp3`, `sfx-victory.mp3`, `sfx-gameover.mp3`
